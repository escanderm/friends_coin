# FriendCoin

Своя криптовалюта для своих. Отправляй монеты друзьям, майни блоки, всё по-настоящему — блокчейн, подписи, Proof-of-Work.

## Как это работает

- Каждый участник получает **RSA-кошелёк** (публичный + приватный ключ)
- Транзакции **подписываются** приватным ключом — подделать нельзя
- Блоки добываются через **Proof-of-Work** (как в Bitcoin, только попроще)
- Награда за майнинг — **10% от суммы транзакций** в блоке (макс 10 монет)
- Можно майнить **пустые блоки** (без транзакций) — награда всего 1 монета
- Максимум **3 транзакции** на блок, выбираются случайно
- Майнер **не получает %** от транзакций, где он участник (защита от накрутки)
- Все клиенты синхронизируются через **WebSocket-сервер**
- Автоматическое **уведомление об обновлениях** — сервер проверяет версию клиента

## Быстрый старт (из исходников)

### 1. Клонируй репозиторий

```bash
git clone <url-репозитория>
cd BLOCKCHAIN
```

### 2. Установи зависимости

```bash
npm install
```

### 3. Запусти клиент

```bash
node start.js
```

При первом запуске программа предложит создать кошелёк — введи своё имя (латиницей, без пробелов). Кошелёк сохранится в файл `wallet_<имя>.json`.

## Команды в клиенте

| Команда              | Что делает                                              |
| -------------------- | ------------------------------------------------------- |
| `balance`            | Показать текущий баланс                                 |
| `send <имя> <сумма>` | Отправить монеты другу                                  |
| `mine`               | Добыть блок (есть транзакции — обычный, нет — пустой)  |
| `friends`            | Список друзей в сети                                    |
| `address`            | Показать свой публичный адрес                           |
| `syncpool`           | Синхронизировать пул транзакций с сервером              |
| `help`               | Справка по командам                                     |
| `exit`               | Выйти                                                   |

## Пример

```
Alice> send Bob 50

Bob> mine
⛏️ Блок #3 добыт за 2.4 секунд!
   Награда: 5 монет (10% от массы, макс 10)

Bob> balance
💰 Баланс: 55 монет
```

## Как это устроено

```
blockchain.js  — ядро: Transaction, Block, Blockchain (SHA-256, RSA, PoW)
wallet.js      — генерация и хранение RSA-2048 ключей
user.js        — клиент: подключение к серверу, CLI, отправка/майнинг
start.js       — точка входа: выбор или создание кошелька
server.js      — WebSocket-сервер: синхронизация цепочки между клиентами
```

## Безопасность кошелька

Файл `wallet_<имя>.json` — это твой приватный ключ. Кто им владеет, тот владеет монетами. Лучший способ хранения — **на флешке или внешнем диске**, отдельно от проекта.

### Как пользоваться кошельком на флешке

При запуске `node start.js` выбери пункт **"Загрузить кошелёк с флешки/диска"** и укажи путь к файлу:

```
Путь к файлу кошелька (wallet_*.json): /Volumes/MYFLASH/wallet_Bob.json
```

Программа загрузит ключи прямо с носителя — копировать файл в папку проекта не нужно.

### Правила

- **Не теряй файл кошелька** — потерял ключ = потерял монеты навсегда
- **Не показывай приватный ключ никому** — с ним можно тратить твои монеты
- **Не коммить кошелёк в git** — файлы `wallet_*.json` уже в `.gitignore`, но будь внимателен
- Друзья появляются в адресной книге автоматически при подключении к серверу
- Чтобы транзакции подтвердились, кто-то должен добыть блок (`mine`)
- Можно майнить и без транзакций — награда будет 1 монета, но сложность выше

## Экономика

| Параметр | Значение |
| --- | --- |
| Начальная эмиссия | 1000 монет (genesis-блок) |
| Награда за блок | 10% от суммы транзакций |
| Потолок награды | 10 монет |
| Пустой блок | 1 монета, та же сложность |
| Макс транзакций в блоке | 3 (случайный отбор) |
| Сложность (PoW) | 7.5 (автоподстройка каждые 50 блоков, цель ~10 мин) |
